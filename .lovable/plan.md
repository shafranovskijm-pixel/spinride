
# План: Автоматическая привязка товаров к категориям

## Обзор

Необходимо привязать **30 товаров без категорий** к соответствующим категориям на основе ключевых слов в названиях.

## Распределение товаров

### 1. Декор (категория `decor`) — 14 товаров
Все ёлки:
- Белая Ель 34DW
- Искусственная Ель мини
- Комбинированная Ель 2326SL
- Литая голубая Ель 2345В
- Литая голубая Ель с гирляндой 2345BL
- Литая Ель 2227, 2253, 2424, 2434, 2435, 2445А, 2445В
- Литая Ель заснеженная 2404S, 2444S
- Литая Ель премиум 2430
- Литая Ель с гирляндой 2405
- Литая искусственная Ель 2444

### 2. Зимние товары (категория `winter`) — 5 товаров
Тюбинги и коньки:
- Коньки детские раздвижные
- Коньки фигурные
- Коньки хоккейные
- Санки-тюбинг надувные
- Тюбинг с пластиковым дном

### 3. Праздничные товары (категория `party`) — 7 товаров
Деды Морозы и новогодние сувениры:
- Дед Мороз в золотой шубе
- Дед Мороз и Снегурочка (синий)
- Дед Мороз музыкальный
- Дед Мороз премиум (серебро/золото)
- Коллекция Дедов Морозов
- Снегурочка музыкальная
- Конфетница новогодняя

### 4. Детям (категория `kids`) — 1 товар
- Дрифт-карт

---

## Технические детали

Выполню 4 SQL-запроса UPDATE для массового обновления `category_id`:

```text
┌─────────────────────────────────────────────────────────────┐
│  Правило                    │  category_id                  │
├─────────────────────────────┼───────────────────────────────┤
│  name LIKE '%Ель%'          │  40b7373f... (decor)          │
│  name LIKE '%Коньки%'       │  3475caa0... (winter)         │
│  name LIKE '%Тюбинг%'       │  3475caa0... (winter)         │
│  name LIKE '%Санки%'        │  3475caa0... (winter)         │
│  name LIKE '%Дед Мороз%'    │  55622af6... (party)          │
│  name LIKE '%Снегурочка%'   │  55622af6... (party)          │
│  name LIKE '%Конфетница%'   │  55622af6... (party)          │
│  name = 'Дрифт-карт'        │  3f590054... (kids)           │
└─────────────────────────────────────────────────────────────┘
```

После выполнения все 30 товаров будут привязаны к соответствующим категориям и станут отображаться в каталоге.
