
# План: Обновление контактных данных

## Проблема

На странице "Контакты" отображаются тестовые/плейсхолдерные данные вместо реальных контактов магазина.

## Текущие данные → Правильные данные

| Поле | Сейчас | Нужно |
|------|--------|-------|
| Телефон | +7 (999) 123-45-67 | +7 924-788-11-11 |
| Адрес | г. Москва, ул. Примерная, д. 1 | г. Уссурийск, ул. Пушкина, 13 |
| WhatsApp | wa.me/79991234567 | wa.me/79247881111 |

## Что нужно сделать

Обновить запись в таблице `page_content` с `page_key = 'contacts'`, изменив поле `content` с правильными контактными данными.

## Результат

После обновления на странице "Контакты" будут отображаться реальные данные магазина SPINRIDE в Уссурийске.

---

## Технические детали

Выполнить SQL-запрос UPDATE для таблицы `page_content`:

```sql
UPDATE page_content 
SET content = jsonb_set(
  jsonb_set(
    jsonb_set(
      content::jsonb,
      '{phone}', '"+ 7 924-788-11-11"'
    ),
    '{address}', '"г. Уссурийск, ул. Пушкина, 13"'
  ),
  '{social,whatsapp}', '"https://wa.me/79247881111"'
)
WHERE page_key = 'contacts';
```
